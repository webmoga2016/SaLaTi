# 🕌 Prayer Times App | تطبيق الأذان

A beautiful bilingual (Arabic/English) Progressive Web App for prayer times, Qibla direction, and Hijri calendar with adjustable offset.

## ✨ Features | المميزات

- 🕐 **Accurate Prayer Times** - Calculated based on your location
- 🧭 **Qibla Compass** - Interactive compass with real-time direction
- 📅 **Dual Calendar** - Gregorian and Hijri dates side by side
- ➕ **Hijri Adjustment** - Adjust Hijri date by ±2 days
- 📊 **Weekly View** - View prayer times for the entire week
- 🌐 **Bilingual** - Full Arabic and English support
- 📱 **PWA Ready** - Install as app on any device
- 🔄 **Offline Support** - Works without internet after first load
- 🔔 **Countdown Timer** - Real-time countdown to next prayer

## 🚀 Deployment

### GitHub Pages
1. Create a new repository on GitHub
2. Upload all files to the repository
3. Go to Settings > Pages
4. Select source as "main" branch
5. Your app will be live at `https://yourusername.github.io/repository-name`

### Convert to APK (Android App)
1. Visit [PWABuilder.com](https://www.pwabuilder.com)
2. Enter your GitHub Pages URL
3. Click "Start"
4. Wait for the analysis to complete
5. Click "Build My Package"
6. Select "Android" platform
7. Download the `.apk` file
8. Install on your Android device
